# Exchange Rates
## Simple Chrome Extension

### Installation
Install latest node.js 

1. pnpm install
2. chrome://extensions/
3. Enable Developer mode
4. Load unpacked - select dist/ folder which contains production-ready build 

### Used tech stack
1. React 18.2
2. Vite 3.1
3. Typescript 4.8
4. React-router-dom
5. Effector 22.7
